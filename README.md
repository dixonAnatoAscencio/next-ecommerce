## Correr en dev
1. Clonar el repo 
2. Crear una copia del ```.env.template``` y renombrarlo a ```.env``` y cambiar las variables de entorno
3. Instalar dependencias ```npm install ```
4. Levantar db ```docker compose up -d```
5. Correr el proyecto ```npm run dev```
